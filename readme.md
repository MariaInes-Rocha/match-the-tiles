Match the tiles

O jogo consiste em colocar todas as peças coloridas nos locais designados com a mesma cor.
De modo a resolver o quebra-cabeças, os movimentos são sincronizados e os espaços entre as peças são criados pela peças fixas existentes. 
Existem vários níveis e portanto diferentes configurações do jogo.

Para executar este jogo será necessário ter instalados o python e o pygame.

Para ter acesso ao jogo é necessário descarregar os ficheiros para uma pasta no diretório de preferência.

Para o iniciar basta executar no terminal no diretório escolhido o comando: python main.py

Se as peças forem colocadas no sítio correspondente o jogo termina e volta automáticamente ao menu. Se pretender reiniciar o jogo basta clicar na tecla 'r'.

Bom jogo!
